Serverul de baze de date utilizat

Pentru a reține toți utilizatorii și datele asociate acestora va fi nevoie de o bază de date MySQL:

  • Tabela user_data: este folosita pentru operatiile de login / inregistrare.
  • Tabela date_personale: contine datele personale ale contului care vor fi introduse si in CV, date comune CV-urilor de orice tip.
  • Tabela educatie: contine datele din categoria Educatie. Are cheia primara formata din: user_id si entry_id, cel din urma fiind folosita pentru distingerea CV-urilor create de acelasi utilizator.
  • Tabela experienta: contine datele din categoria Experienta. Are cheia primara formata din: user_id si entry_id, cel din urma fiind folosita pentru distingerea CV-urilor create de acelasi utilizator.
  • Tabela abilitati: contine datele din categoria Abilitati.
  • Tabela limbi_straine: contine datele din categoria Limbi straine.
  • Tabela alt_info: contine experienta in domeniu a fiecarui utilizator sub forma de internship-uri / stagii de practica.
  • Tabela permis: are cheia primara formata din: user_id si entry_id, cel din urma fiind folosita pentru distingerea permisurilor de conducere detinute de acelasi utilizator.
